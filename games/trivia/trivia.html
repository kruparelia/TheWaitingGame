BP Trivia

Levels:
  • Level 1 - Easy - A list of answers is displayed for each question. Click the correct answer.
  • Level 2 - Difficult - A text box is displayed in which you must write the answer, then click on Send button.
Mode
  • Consecutive - The questions start from the quiz with index number added into a text field, and are added in their order till the last question.
  • Random - The questions are choosed randomly, without repeat.
• Countdown Timer - If that button is checked, you have 15 seconds to answer till the next question is shown.

- Click the Start button to start the quizzes. The Reset button resets the Trivia.
